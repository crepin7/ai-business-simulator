# 🏢 AI Business Simulator

Jeu de gestion d'entreprise révolutionnaire avec intelligence artificielle pour la prise de décision, l'analyse de marché et la planification stratégique.

## ✨ Fonctionnalités principales

- **Décisions IA** : IA pour guider vos décisions stratégiques
- **Analyse de marché** : Analyse dynamique du marché concurrentiel
- **Planification stratégique** : Stratégies basées sur l'IA
- **Économie dynamique** : Système économique intelligent
- **Concurrents IA** : Concurrents avec comportements réalistes
- **Scénarios réels** : Scénarios basés sur le monde réel
- **Multiplateforme** : Compatible avec PC, mobile et web

## 🛠️ Stack technologique

- **Game Engine** : Custom engine en JavaScript/TypeScript
- **IA Générative** : GPT-4 pour la prise de décision
- **Analyse de données** : Machine Learning pour l'analyse de marché
- **Graphismes** : Canvas API + WebGL
- **Audio** : Web Audio API pour des effets sonores
- **Multiplateforme** : Web, mobile, desktop

## 📦 Installation

```bash
# Clonez le dépôt
git clone https://github.com/crepin7/ai-business-simulator.git
cd ai-business-simulator

# Installez les dépendances
npm install

# Lancez le jeu
npm start
```
